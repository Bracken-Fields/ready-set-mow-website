# Ready Set Mow — Website UI Kit

Marketing site for ReadySetMow.co. Single-page homepage prototype + reusable JSX components.

## Files
- `index.html` — fully assembled responsive homepage prototype
- `Header.jsx` — sticky nav with horizontal lockup
- `Hero.jsx` — full-bleed hero with badge + CTAs
- `TrustStrip.jsx` — 5 quick trust points
- `Services.jsx` — 6-card service grid
- `WhyUs.jsx` — value-prop block
- `ServiceAreas.jsx` — Anderson / Noblesville / Fishers cards
- `Process.jsx` — 3-step flow
- `SeasonalCTA.jsx` — full-bleed CTA band
- `Testimonials.jsx` — 3 quotes
- `FAQ.jsx` — accordion
- `QuoteForm.jsx` — 9-field quote form
- `Footer.jsx` — logo, areas, contact
- `MobileStickyCTA.jsx` — bottom-fixed bar on mobile
- `Icon.jsx` — Lucide-style line icons (inline SVG, brand stroke)

Components export to `window.RSM` so they're available across `<script type="text/babel">` blocks.
